

## Consulta de Documento e Segmento
```sql
SELECT doc.DESCDOC as Document, ID_PRJ_SEGUIMENTO_MERCADO.NOME as Segment

FROM DDOC doc
         JOIN ID_PRJ_PROJETO projeto ON SUBSTRING(IDDESCDOC, 7, 9) = SUBSTRING(projeto.CODIGO, 5, 9)
         JOIN ID_PRJ_CONTRATO ON projeto.IDCONTRATO = ID_PRJ_CONTRATO.ID
         JOIN ID_PRJ_SEGUIMENTO_MERCADO ON ID_PRJ_CONTRATO.IDSEGUIMENTOMERCADO = ID_PRJ_SEGUIMENTO_MERCADO.ID
        WHERE DOC.DTULTIMAALTERACAO <= '2025-01-28'

```
